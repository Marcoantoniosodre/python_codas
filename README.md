## 🐍 Sobre este repositório

Este repositório é um registro honesto da minha jornada aprendendo Python. Aqui você vai encontrar desde os primeiros códigos confusos até as soluções que finalmente fizeram sentido. Nada foi apagado para parecer mais bonito — faz parte do processo.

- Nunca tinha programado antes

- Comcei por insetinvos de uns amigos meus, com o objetivo de lucrar

## 📚 Tópicos que já estudei

- **Fundamentos**: variáveis, tipos de dados, operadores
- **Estruturas de controle**: `if`, `elif`, `else`, loops `for` e `while`
- **Funções**: definição, parâmetros, retorno, escopo
- **Listas, tuplas, dicionários e sets**
- **Manipulação de strings**
- **Tratamento de erros**: `try`, `except`, `finally`
- **Módulos e importações**
- **Leitura e escrita de arquivos**
- **Introdução à Programação Orientada a Objetos (POO)**

## ❌ Erros que cometi (e aprendi com eles)

> Esses são reais. Coloco aqui porque errar faz parte.

- Confundir `=` com `==` nas condições
- Esquecer de retornar valores em funções
- Usar variáveis fora do escopo sem entender o motivo do erro
- Tentar pular etapas antes de entender o básico — isso atrasou muito
- Não ler as mensagens de erro com calma (o Python quase sempre diz exatamente o que está errado)
